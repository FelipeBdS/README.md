[![NPM](https://img.shields.io/npm/l/react)](https://github.com/FelipeBdS/README.md/blob/main/LICENSE)

# Sobre o Projeto
MegaBass é uma loja virtual voltada para os amantes de contrabaixo de qualidade. Seu intuito é disponibilizar os melhores modelos das maiores marcas do mercado atual.

Primeiro projeto Institucional em desenvolvimento do #XP38 da [Gama Academy](https://www.gama.academy/)
 para fins educativos.
 
 ## Sprint 1: 14/07/2021 a 15/07/2021 - Planejamento
 - Planejamento do site;
 - Pesquisa dos produtos a serem vendidos;
 - Informações gerais dos produtos;
 - Definição da estética do site.


## Sprint 2: 16/07/2021 a 23/07/2021 - Desenvolvimento
- Iniciar pelo cabeçalho e rodapé da página;
- Em seguida o menu de opções, onde estarão disponíveis as marcas dos contrabaixos;
- Página inicial com os destaques dos melhores modelos;
- Desenvolvimento das páginas de cada instrumento;
- Responsividade da página.


## Tecnologias utilizadas
- HTML;
- CSS;
- Bootstrap.
